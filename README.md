# Library Management System

## Project Description

This project is a Library Management System developed using MySQL. It helps manage books, students, and book issue records in a library database.

## Database Tables

### Books

Stores information about books available in the library.

### Students

Stores student details.

### IssueBooks

Stores records of books issued to students.

## Features

* Add new books
* Add student records
* Issue books to students
* Maintain library data
* Manage book records efficiently
